Hotel Halm
Dokumentation

Vollständige Anleitung zur Hotel-Halm-Website. Hier finden Sie alles, was Sie brauchen, um neue Komponenten, Seiten und Elemente hinzuzufügen — inklusive kopierfertiger Code-Snippets.

Architektur

Die Website besteht aus 11 eigenständigen HTML-Dateien, die alle gleich aufgebaut sind. Jede Seite enthält vollständiges CSS, Navigations-HTML, Seiten-Inhalt, Footer und JavaScript — alles in einer Datei.

Stack

Rein HTML / CSS / JS

Kein Framework, kein Build-System. Jede Seite ist eine einzelne .html-Datei. Sofort im Browser öffnen, sofort editieren.

Fonts

Google Fonts (2 Familien)

Cormorant Garamond für Überschriften und Display-Text. Jost für Body-Text, Navigation und Labels.

Bilder

Unsplash CDN

Alle Bilder kommen von images.unsplash.com. Einfach URL austauschen für eigene Fotos — selbes Format beibehalten.

Stil

Dark Luxury Aesthetic

Dunkler Hintergrund, Gold-Akzente, großzügiger Weißraum, Serif-Typografie — inspiriert von Astris Residence / Vide Infra.

Tipp Öffnen Sie jede .html-Datei im Browser mit Doppelklick oder per lokalem Webserver. Für Änderungen einfach im Texteditor (VS Code empfohlen) öffnen, speichern, Browser neu laden.

/* CSS (bereits vorhanden) */
.reveal {
  opacity: 0;
  transform: translateY(36px);
  transition: all 0.9s cubic-bezier(0.25,0.46,0.45,0.94);
}
.reveal.visible {
  opacity: 1;
  transform: translateY(0);
}

/* JS (bereits am Ende jeder Seite) */
const obs = new IntersectionObserver((entries) => {
  entries.forEach(e => {
    if (e.isIntersecting) {
      e.target.classList.add('visible');
      obs.unobserve(e.target);
    }
  });
}, { threshold: 0.1 });
document.querySelectorAll('.reveal').forEach(el => obs.observe(el));

<script>
function toggleAccordion(btn) {
  const body = btn.nextElementSibling;
  const icon = btn.querySelector('.accordion-icon');
  const isOpen = body.style.maxHeight;
  // Alle schließen
  document.querySelectorAll('.accordion-body').forEach(b =>
    b.style.maxHeight = '');
  document.querySelectorAll('.accordion-icon').forEach(i =>
    { i.textContent='+'; i.style.transform=''; });
  // Aktuelles öffnen wenn es zu war
  if (!isOpen) {
    body.style.maxHeight = body.scrollHeight + 'px';
    icon.textContent = '×';
    icon.style.transform = 'rotate(45deg)';
  }
}
</script>

Neue Seite anlegen

Schritt-für-Schritt-Anleitung zum Erstellen einer neuen Seite im exakt gleichen Stil wie alle anderen.

1

Vorhandene Seite als Vorlage kopieren

Nehmen Sie z.B. hotel-halm-konstanz.html und speichern Sie sie als hotel-halm-NEUERSEITE.html. Diese Datei enthält bereits alle nötigen CSS-Variablen, Fonts, Navigation und Footer.

2

<title> anpassen

Ändern Sie in der neuen Datei ganz oben: <title>SEITENNAME — Hotel Halm Konstanz</title>

3

Aktiven Nav-Link markieren

Fügen Sie dem entsprechenden Nav-Link die Klasse active hinzu: <a href="..." class="active">Neue Seite</a> — das unterstreicht den Link gold.

4

Hero-Bild und -Titel ersetzen

Passen Sie den page-hero-bg Hintergrund-URL, das page-hero-label und den page-hero-title an Ihre neue Seite an.

5

Alten Inhalt löschen, neuen einfügen

Löschen Sie alles zwischen Hero-Section und Footer. Fügen Sie Ihre neuen Sektionen ein — verwenden Sie die Code-Snippets aus dieser Dokumentation.

6

Link in allen anderen Seiten ergänzen

Öffnen Sie alle 11 HTML-Dateien und fügen Sie den Link zur neuen Seite in die <ul class="nav-links">-Liste ein.
